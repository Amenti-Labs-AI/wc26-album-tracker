# App branding

- `amenti_logo_mark.svg` — Amenti Labs mark from [amenti-website/public/brand](https://github.com/Amenti-Labs-AI/amenti-website/tree/main/public/brand).
- `amenti_logo_mark.png` — in-app settings logo (256px).
- `app_icon.png` / `app_icon_foreground.png` — launcher icons generated by `tooling/generate_app_icon.py`.

Regenerate after updating the mark:

```bash
python3 tooling/generate_app_icon.py
dart run flutter_launcher_icons
```
